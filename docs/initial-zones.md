# Первичная загрузка зон

Откройте `https://krotka.46.225.65.120.sslip.io`, зайдите как админ и перейдите во вкладку `Зоны`.

Для каждой физической зоны с бумажной схемы заведите отдельную запись:

| Поле | Что вводить |
| --- | --- |
| `Код` | Короткий стабильный код со схемы: `W1`, `W2`, `M1`, `A1` |
| `Название` | Человеческое описание: `Туя у парковки`, `Самшит большой`, `Газон у въезда` |
| `Тип` | `Газон`, `Кустарники`, `Новые посадки`, `Дерево`, `Живая изгородь`, `Другое` |
| `Солнце` | `Солнце`, `Полутень`, `Тень`, `Смешано` |
| `Заметки` | Любые уточнения: расположение, размер, проблемы, номер со схемы |

Главное правило: `Код` должен быть уникальным и не должен меняться. Название можно уточнять позже.

## Черновик по фото

По схеме `20260629_184717.jpg` я бы начал с такого списка, а потом поправил названия в интерфейсе:

| Код | Название | Тип | Заметки |
| --- | --- | --- | --- |
| `W1` | Туя / зона W1 | Дерево | уточнить номер со схемы |
| `W2` | Самшит / зона W2 | Кустарники | уточнить номер со схемы |
| `W3` | Туя / зона W3 | Дерево | уточнить номер со схемы |
| `W4` | Самшит большой / зона W4 | Кустарники | справа внизу на схеме |
| `M1` | Туя / зона M1 | Дерево | уточнить расположение |
| `M2` | Самшит / зона M2 | Кустарники | уточнить расположение |
| `M3` | Туя / зона M3 | Дерево | около центральной области |
| `A1` | Туя / зона A1 | Дерево | нижняя/левая зона по схеме |
| `A2` | Туя / зона A2 | Дерево | левая верхняя зона по схеме |
| `A3` | Посадки A3 | Другое | верхняя центральная зона |
| `A4` | Посадки A4 | Другое | верхняя правая зона |

Если одна буква на схеме относится не к растению, а к группе полива, используйте отдельные коды вроде `W1-1`, `W1-2`, `W1-3`.
